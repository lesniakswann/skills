# Installing A Skill In Claude
*A quick guide for the whole team*

## What Is A Skill?

Skills are small instruction files that help Claude handle a particular type of work. For example, a skill might help Claude format reports in the Lesniak Swann house style.

Once installed, a skill works quietly in the background. You do not need to mention it every time. Claude applies it automatically when it is relevant.

## Where To Find Skills

Use the pinned Slack message in **#claude-skills**.

That message lists the available skills and links directly to each one, clicking a link will download the skill directly.

## Before You Start

You will need:

- Access to the pinned **#claude-skills** Slack message
- A Claude account, either on claude.ai or in the Claude desktop app
- A couple of minutes

## Step 1 - Download The Skill

1. **Open the pinned Slack message**
   Go to **#claude-skills** and open the pinned message that lists the available skills.

2. **Choose the skill**
   Click the link for the skill you want to install. The skill file will download directly.

3. **Find the downloaded file**
   Look in your Downloads folder, or wherever your browser usually saves downloaded files.

When screenshots are added to this guide, follow the highlighted links and buttons shown in the screenshots.

## Step 2 - Add The Skill To Claude

The steps are slightly different depending on whether you use Claude on the web or in the desktop app, but the process is the same.

### On The Web

1. **Open Claude**
   Go to claude.ai and sign in if needed.

2. **Open Customise**
   Click your profile icon or name in the top-right corner, then choose **Customise**.

3. **Go to Skills**
   In the Customise panel, look for the **Skills** section.

4. **Add the skill**
   Click **Add skill**, then choose **Upload file**. Select the skill file you downloaded.

5. **Done**
   The skill is now active. You do not need to restart anything.

### In The Desktop App

1. **Open Claude**
   Launch the Claude desktop app.

2. **Open Customise**
   Click your profile icon or name, then choose **Customise**.

3. **Go to Skills**
   Find the **Skills** section.

4. **Add the skill**
   Click **Add skill**, then **Upload file**. Select the skill file you downloaded.

5. **Done**
   The skill is installed and will apply to future conversations.

## Updating A Skill

When a skill is updated, replace your installed version with the latest file from the pinned Slack message.

1. **Open the pinned Slack message**
   Go back to **#claude-skills** and click the link for the skill.

2. **Download the latest version**
   The latest skill file will download directly.

3. **Open Skills in Claude**
   Open **Customise**, then go to **Skills**.

4. **Replace the old version**
   Find the skill in your Skills list, open its three-dot menu, choose **Replace**, and upload the new file.

5. **Done**
   The updated skill takes effect immediately.

## Need Help?

Post in **#claude-skills** if something is not working, if you cannot see the Skills section in Claude, or if you want to suggest a new skill.
