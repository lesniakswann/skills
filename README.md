# Lesniak Swann Skills

This is the Lesniak Swann repository for reusable AI skills. Each folder contains a skill that can be added to compatible assistants, such as Claude, to give them project-specific guidance, brand rules, workflows, or reference material.

More information can be found in the Lesniak Swann Slack channel **Claude Skills**.

## Available Skills

### Lesniak Swann Document Brand Style

Folder: `lesniak-swann-document-brand-style`

Applies the Lesniak Swann document brand style to DOCX, PDF, slides, proposals, reports, exports, and styled Markdown, including typography, spacing, hierarchy, brand colours, tables, charts, and document-first output guidance.

Example uses:

- "Create a PDF proposal from this outline in the Lesniak Swann document style."
- "Turn these notes into a polished DOCX using the Lesniak Swann style."
- "Restyle this draft so it feels like a Lesniak Swann client document."
- "Create a short slide deck from this summary using the Lesniak Swann document brand style."
- "Format this strategy document as styled Markdown for Lesniak Swann."

## Downloading A Skill

1. Open the folder for the skill you want to use.
2. Download the skill.md file.
3. If your browser downloads it as a zip file, unzip it before adding it to Claude.

## Adding A Skill To Claude

In Claude Desktop or Claude on the web:

1. Open **Customize**.
2. Choose **Add skill**.
3. Select or upload the downloaded skill folder.
4. Save the change.

Once added, Claude can use the skill when the task matches its description.
