# Lesniak Swann Skills

This is the Lesniak Swann repository for reusable AI skills. Each folder contains a skill that can be added to compatible assistants, such as Claude, to give them project-specific guidance, brand rules, workflows, or reference material.

More information can be found in the Lesniak Swann Slack channel **Claude Skills**.

## Available Skills

### Lesniak Swann Report Style

Folder: `lesniak-swann-report-style`

Applies the Lesniak Swann report style to DOCX, PDF, slides, and styled Markdown, including typography, spacing, hierarchy, brand colours, logo use, tables, charts, and document-first output guidance.

Example uses:

- "Create a PDF report from this research in the Lesniak Swann report style."
- "Turn these notes into a polished DOCX report using the Lesniak Swann style."
- "Restyle this draft so it feels like a Lesniak Swann client report."
- "Create a short slide deck from this summary using the Lesniak Swann report style."
- "Format this strategy document as a styled Markdown report for Lesniak Swann."

## Downloading A Skill

1. Open the folder for the skill you want to use.
2. Download the skill.md file.
3. If your browser downloads it as a zip file, unzip it before adding it to Claude.

## Adding A Skill To Claude

In Claude Desktop or Claude on the web:

1. Open **Customize**.
2. Choose **Add skill**.
3. Select or upload the downloaded skill folder.
4. Save the change.

Once added, Claude can use the skill when the task matches its description.
